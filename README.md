## FourCast 🌦️
A modern, API-driven weather application featuring real-time data, geolocation, and dynamic UI rendering.

## 🧩 Overview
A modern weather forecast web app built with JavaScript, integrating multiple APIs for real-time data and location-based forecasts. Beyond a simple weather app, it focuses on dynamic background changes and more advanced UI design to create a richer and more engaging user experience.

## 🎯 Project Goals
The goal of this project was to deepen my understanding of core JavaScript by building a dynamic application without relying on frameworks. I focused on structuring functions with clear responsibilities, handling asynchronous data, and managing real-time updates through DOM manipulation. I also challenged myself to design a more complex and refined UI using advanced HTML and CSS, including Flexbox and Grid layouts.

## 🛠 Tech Stack
- HTML5  
- CSS3 
- JavaScript 

## ✨ Main Features
- Real-time weather data from multiple APIs (Open-Meteo, OpenStreetMap)
- Geolocation-based forecasting and city search
- Dynamic UI rendering based on weather conditions and time of day
- Weather code mapping to icons and background videos
- Hourly and weekly forecast visualization
- Responsive UI with scroll indicators and adaptive layout
- Sunrise and sunset visualization with time-based logic
- Error handling for API failures and location permissions

## ⚠️ Challenges I Faced
- Managing asynchronous data flow
- Structuring and filtering complex API response data
- Implementing dynamic UI logic based on time and weather conditions
- Maintaining code organization in a large, DOM-driven application
  
## 🤖 How I Used AI

- Provided guidance on handling API errors and edge cases
